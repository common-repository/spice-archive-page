Plugin allows you to display yearly, monthly and daily archives in pages.

== Description ==

Spice Archive page plugin allows you to display yearly ,monthly and daily archives in pages and posts. Just add the shortcode *[spicearchive]*.
Archives can be displayed using a shortcode in post, pages and sidebars.

#### Features
* Yearly Archive
* Monthly Archive
* Daily Archive
* Limit number of results to show in archive.

#### Roadmap
* We will add features as per user feedback.

#### Live Demo
[https://spicewp.com/spice-archive-page/](https://spicewp.com/spice-archive-page/)

== Installation ==

1. Download Spice Archive Page plugin.
2. Unzip and upload the spice-archive-page folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress and Enjoy.
